Завдання №1

В Figma створити макет галереї довільних зображень.

Виконати верстку власного макету. В HTML коді (index.html ) додати тег контейнера галереї — невпорядкований список із класом gallery, згідно макету.

У свій JavaScript файл додай масив зображень. Кожний об’єкт являє собою один елемент галереї.

Виконати делегування, а саме додати функціонал прослуховування кліка по елементах галереї та отримання посилання на велике зображення при кліку. Для цього використовуй прийом делегування на ul.gallery. Поки що при кліку на елемент галереї виводь у консоль посилання на велике зображення.

Доповнити свій код так, щоб при кліку по елементу галереї відкривалось модальне вікно підключеної бібліотеки.
