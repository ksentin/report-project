Висновки:

Під час виконання завдань успішно використала об'єкти, їх методи та властивості в JavaScript. Практикувалася у використанні функцій зворотного виклику (callback functions) для обробки подій, таких як натискання на кнопки чи відправлення форм. Використання стрілочних функцій дозволило зробити код більш компактним та зрозумілішим, особливо при роботі з подіями та ітераціями.

В процесі проробленої роботи реалізовано кілька функціональностей за допомогою JavaScript та CSS:

  • У JavaScript файлі (gallery.js) створено масив об'єктів з даними зображень.
  • Кожен об'єкт містить посилання на маленьку версію зображення (preview), велику версію зображення (original), і текстовий опис (description).
  • Виконано динамічну розмітку елементів галереї та додано їх у ul class="gallery">.li
  • Реалізовано прослуховування події click на елементах галереї з використанням делегування.
  • При кліку на елемент галереї виводиться у консоль посилання на велике зображення.
  • Підключено бібліотеку basicLightbox за допомогою CDN сервісу jsdelivr. Реалізовано відкриття модального вікна з великим зображенням при кліку на елемент галереї.
  • У JavaScript файлі (2-form.js) оголошено об'єкт formData з полями email та message, які спочатку мають порожні рядки. Реалізовано делегування подій для відстеження змін у формі через подію input. Дані з полів email та message зберігаються у локальне сховище під ключем feedback-form-state.

Реалізовано необхідний функціонал для збереження стану форми та відкриття модальних вікон для галереї зображень. Таким чином, завдання виконано повністю та відповідає вимогам.
