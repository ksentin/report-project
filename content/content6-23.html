Висновки:

Під час виконання завдань успішно використала об'єкти, їх методи та властивості в JavaScript. Практикувалася у використанні функцій зворотного виклику (callback functions) для обробки подій, таких як натискання на кнопки чи відправлення форм. Використання стрілочних функцій дозволило зробити код більш компактним та зрозумілішим, особливо при роботі з подіями та ітераціями.

В процесі проробленої роботи реалізовано кілька функціональностей за допомогою JavaScript та CSS:

  • Реалізація кнопки, яка міняє місцями значення двох полів вводу.
  • Розробка скрипта, який змінює розмір квадрата в залежності від натискання на кнопки "Збільшити" та "Зменшити".
  • Створення кнопки, яка подвоює значення усіх елементів списку.
  • Розробка скрипта для обробки списку категорій, який підраховує кількість категорій та кількість елементів у кожній категорії.
  • Реалізація форми логіна, яка перевіряє заповненість полів та виводить їх значення у консоль.
  • Розробка скрипту для зміни кольору фону елемента та виведення випадкового кольору у спані.
  • Створення кнопок для створення та видалення колекції елементів, яка може розтягуватися у висоті та містити різні кольорові блоки.

Кожен елемент був реалізований окремим скриптом та містив відповідну функціональність, забезпечуючи коректну роботу згідно з вимогами завдань. Також застосовано CSS для стилізації елементів та їх розташування на сторінці.
